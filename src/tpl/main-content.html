Registrazione repository per software pubblico

Developers Italia, al fine di popolare il catalogo del software pubblico, effettua scansioni periodiche sugli strumenti di code hosting dichiarati dalle amministrazioni per rilevare i repository contenenti un file publiccode.yml.

I tre campi sottostanti verranno automaticamente compilati con i risultati della ricerca.
